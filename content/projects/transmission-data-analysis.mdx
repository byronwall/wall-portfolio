Built a 100k+ LOC platform using C#, TypeScript, and React for analyzing transmission testing data.

## Key Features

- Custom calculation DSL/syntax for complex data analysis
- Interactive visualizations using custom charting components (SVG + canvas)
- Real-time data processing and filtering
- Multi-user support with role-based access control

## Technical Details

The platform consists of:

- Backend API built with C# and .NET
- Frontend SPA using React and TypeScript
- Custom charting library built on D3.js
- SQLite database for data storage
