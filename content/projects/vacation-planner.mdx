Family Vacation Planner is a map-first trip planning app for comparing lodging options across cities and candidate date ranges. It is built for the messy middle of family travel planning: tracking options, dates, notes, party size, listing links, and fit across possible weeks before anything is booked.

- Source: [byronwall/vacation-planner](https://github.com/byronwall/vacation-planner)

## Workflow

The planner organizes travel decisions around trip workspaces:

- Create trips with notes and party size.
- Track multiple cities or locations per trip.
- Compare candidate date ranges with weekly summaries.
- Store lodging options with statuses like `considering`, `shortlisted`, `rejected`, and `booked`.
- Record date-fit, per-range costs, and notes for each lodging option.
- Filter the map by location, status, and date-fit while reviewing options.

## Capture And Backup

The repo includes a Chrome extension for quick listing capture from the active browser tab. Captured listing content can be saved into the local planner, then optionally processed with AI-assisted structured extraction.

Planner data is stored locally and can be exported or imported as a ZIP backup from the app UI.

## Highlights

- SolidStart app with Ark UI, Park UI, and Panda CSS
- Map-centered lodging comparison workflow
- Listing deep links and quick platform search links
- Admin data editor for direct record management
- Optional browser extension for listing capture
- Docker Compose setup with persistent planner data
