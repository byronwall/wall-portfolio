Decisory is an AI-powered consultation tool for working through complex decisions with structured question-and-answer sessions. Instead of starting from a blank page, users describe a goal, answer targeted follow-up questions, and receive recommendations based on the accumulated context.

## Workflow

Decisory turns a loose goal into a guided consultation:

- Start a session by describing the decision or project goal in plain language.
- Generate a concise session title, description, and initial question set.
- Answer multiple-choice questions with optional custom context.
- Review AI-generated recommendations based on submitted answers.
- Run additional rounds when the decision needs refinement.
- Copy analysis results as Markdown for reuse elsewhere.

## Highlights

- 8-12 targeted questions generated for each situation
- Iterative follow-up rounds that build on previous answers
- Local JSON session persistence
- Collapsible question sections and expandable long prompts
- SolidStart, Panda CSS, Ark UI, and OpenAI-backed server logic
