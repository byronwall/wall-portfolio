## Tech Stack

This site is built with:

- Next.js with App Router
- MDX and Markdown for content
- Tailwind CSS
- SEO optimizations (sitemap, robots.txt, JSON-LD schema)
- RSS Feed
- Dynamic OG images
- Syntax highlighting
- Vercel hosting and analytics

## Local Development

Run the development server:

```bash
pnpm dev
```

Build and start:

```bash
pnpm build
pnpm start
```
